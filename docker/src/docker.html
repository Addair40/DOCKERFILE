QUE ES DOCKERFILE

Un DockerFile es un documento de texto que contiene todos los comandos que queramos ejecutar en la línea de comandos para armar una imagen, esta imagen se creará mediante el comando docker build que irá siguiendo las instrucciones.Dockerfile es un documento de texto que contiene las órdenes a las que un usuario puede llamar desde la línea de comandos para crear una imagen. Si hacemos un apt install php, ese mismo comando podemos ejecutarlo desde Dockerfile. Las imágenes pueden almacenarse localmente o remotamente un contenedor encapsula el proceso en ejecución y lo aísla del resto, como si de una máquina virtual se tratase esto permite optimizar el uso de recursos y simplifica la gestión de los mismos es posible crear estos contenedores con una instancia de linux corriendo a través de las tecnologías: namespaces, cgroups y chroot. Docker es un gran aliado, el cual nos proporciona una serie de herramientas que nos van a ayudar a diseñar estos contenedores de forma muy eficiente.
